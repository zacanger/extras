# markdown-powered knowledgebase

## how to use:

* `npm i`
* `mkdir content`
* `cd content`
* `touch foo.md`
* then put stuff in there.
* `npm start`
* `content` can contain an `UPLOADS` subdirectory; this isn't indexed. it's for static files n shiz.
