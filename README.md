# asperitas

Initial code based on [Asperitas](https://github.com/d11z/asperitas).

`docker run -d -p 27017:27107 mongo`
`cd server && npm ci && npm run dev`
`cd client && npm ci && npm start`

## Testing

`docker run -d -p 27017:27107 mongo`
`cd server && npm test`
`cd client && npm test`
